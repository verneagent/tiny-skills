# Agents Guide

This repository contains user-level skills for AI coding agents.

## Rules

- When adding, removing, or renaming a skill, update `README.md` to keep the skill table and install instructions current.
- All skill edits happen in this repo first, then sync to `~/.agents/skills/` via `adhoc-skill sync`.
- Each skill directory must contain a `SKILL.md` with valid frontmatter (`name`, `description`).
